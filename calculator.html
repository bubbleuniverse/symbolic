Quick Start

  1. Enter a mathematical expression in the input field above
  2. Select an operation (integrate, differentiate, etc.) from the dropdown
  3. Click "Calculate" to see the result

Examples

Integration: x^2 → x³/3
Differentiation: x³ → 3x²
Series: sin(x) → x - x³/6 + x⁵/120 + ...

Important Notes

  • Only use 'x' as a variable
  • a, b, c are real constants
  • Use exp(x) instead of e^x
  • First calculation may take up to 15 seconds
